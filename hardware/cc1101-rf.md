# CC1101 — Sub-GHz RF Transceiver

> Based on official Texas Instruments documentation.  
> Source: https://www.ti.com/product/CC1101

---

## Overview

The CC1101 is a low-power Sub-1 GHz RF transceiver manufactured by Texas Instruments, designed for low-power wireless applications. It integrates an RF transceiver with a configurable baseband modem.

---

## Frequency Bands

| Band   | Frequencies   |
|--------|---------------|
| Band 1 | 300 – 348 MHz |
| Band 2 | 387 – 464 MHz |
| Band 3 | 779 – 928 MHz |

Common ISM/SRD frequencies: **315 MHz, 433 MHz, 868 MHz, 915 MHz**

> ⚠️ Actual usable frequencies depend on regional regulations and antenna design.

---

## Key Specifications

| Parameter              | Value                                              |
|------------------------|----------------------------------------------------|
| Data rate              | Up to 600 kbps                                     |
| Max output power       | Up to +12 dBm (depending on frequency and config)  |
| RX current             | 14.7 mA @ 1.2 kBaud, 868 MHz                      |
| Sensitivity            | Up to -116 dBm (configuration dependent)           |
| Interface              | SPI                                                |
| TX/RX FIFO             | 64 bytes each                                      |
| Frequency synth settle | 75 µs                                              |
| Supply voltage         | 1.8 – 3.6 V                                       |

## Modulation Formats

- 2-FSK
- GFSK
- ASK / OOK
- 4-FSK
- MSK

---

## Key Features

- Programmable output power for all supported frequencies
- Configurable baseband modem
- On-chip support for sync word detection, address check, flexible packet length, and automatic CRC handling
- Clear Channel Assessment (CCA)
- Link Quality Indication (LQI)
- Wake-on-radio support
- Automatic Frequency Compensation (AFC)
- Fast frequency hopping enabled by a 75 µs synthesizer settling time
- Configuration and status registers accessible through SPI burst transfers

---

## Typical Applications

- Remote controls
- Wireless sensors
- Smart meters
- Home automation
- Alarm systems
- Industrial telemetry
- Custom Sub-GHz protocols

---

## Notes

- The CC1101 is not LoRa-compatible.
- The CC1101 is not a Wi-Fi or Bluetooth radio.
- Communication protocols must be implemented on top of the supported modulation schemes.
