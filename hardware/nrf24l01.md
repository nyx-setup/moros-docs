# nRF24L01 — 2.4 GHz RF Transceiver

> Based on official Nordic Semiconductor documentation.  
> Source: https://www.mouser.com/datasheet/2/297/nRF24L01_Product_Specification_v2_0-9199.pdf

---

## Overview

The nRF24L01 is a single-chip 2.4 GHz RF transceiver manufactured by Nordic Semiconductor. It operates in the worldwide ISM band and includes an embedded baseband protocol engine (Enhanced ShockBurst™), designed for ultra low-power wireless applications.

---

## Key Specifications

| Parameter          | Value                          |
|--------------------|--------------------------------|
| Frequency          | 2.400 – 2.4835 GHz             |
| Max data rate      | 2 Mbps                         |
| Output power       | Up to 0 dBm (chip specification)|
| Interface          | SPI                            |
| Supply voltage     | 1.9 – 3.6 V                   |
| TX current         | 11.3 mA @ 0 dBm               |
| RX current         | 12.3 mA @ 2 Mbps              |
| Power down current | 900 nA                         |
| Standby current    | 22 µA                          |

---

## RF Channels

- 125 configurable channels in the 2.4 GHz ISM band
- Channel spacing: 1 MHz
- Can be used to implement frequency hopping through software-controlled channel switching

---

## Typical Range

- Indoor: 10 – 30 m
- Outdoor (line of sight): 50 – 100 m

> Note: PA/LNA external amplifier modules can significantly extend range.

---

## Enhanced ShockBurst™

Enhanced ShockBurst™ is the built-in packet-based data link layer. It handles:

- Automatic packet assembly and timing
- Automatic acknowledgement (ACK)
- Automatic retransmission
- CRC error detection
- Dynamic payload length (1 – 32 bytes)

This allows reliable communication without complex protocol implementation on the host microcontroller.

---

## Enhanced ShockBurst™ Limitations

- Maximum payload size: 32 bytes
- Proprietary Nordic protocol
- Not interoperable with Wi-Fi, Bluetooth, Zigbee, or Thread

---

## Key Features

- Operates in the globally available 2.4 GHz ISM band
- Up to 6 logical RX data pipes (MultiCeiver™)
- Configurable output power
- Programmable LNA gain for RX current optimization
- All configuration via SPI register map
- IRQ pin for interrupt-driven operation

---

## Typical Applications

- Wireless keyboards and mice
- Remote controls
- Wireless sensors
- Home automation
- Gaming controllers
- Custom 2.4 GHz protocols

---

## Notes

- The nRF24L01 is **not** Wi-Fi or Bluetooth compatible
- It operates on 2.4 GHz but uses a proprietary protocol — not interoperable with Wi-Fi or BLE devices
- The T-Embed CC1101 Plus includes the nRF24L01 module, but GPIO pin assignments for the K268-01 variant are not yet publicly documented by LilyGO
